# Configuration

v2 has brought a few breaking changes on the configuration. To see how to configure the libary please refer to the configuration section.
Callbacks have been removed and there is no plan on adding them back at the moment.

# API

The most notable change from v1 is the separation of the old screen object into 2 distinct objects, screen and grid. You can check their properties in the API reference
